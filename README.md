Development
===========


To start a new development env, run the following:

1.  rails s
2.  yarn
3.  ./bin/webpack-dev-server



Dependencies

1.  elasticsearch
2.  minimagic
3.  aws-sdk
4.  webpack (bundler for vuejs)
5.  postgresql
6.  graphql
